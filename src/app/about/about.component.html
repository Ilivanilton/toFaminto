Sobre

TôFamito | Uma aplicação em Angular.

O que é?

É uma praça de alimentação virtual, comparada a Ifood, UberEats etc..

Qual Propósito?

Este projeto é destinado a demonstrar minha( @ilivanilton ) evolução profissional no desenvolvimento de aplicações SPA utilizando o framework Angular e o uso da plataforma GitHub, bem como a disseminação do conhecimento técnico por meio de código compartilhado. Logo, Pull Requests serão aceitos.
